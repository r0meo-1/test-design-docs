## Разрешения / адаптив
- [ ] 360–414px (мобильные)
- [ ] 768–1024px (планшеты)
- [ ] 1280–1920px (десктоп)
- [ ] Нет горизонтального скролла и наложений элементов

## Проверяемое на каждой связке
- [ ] Каталог и карточка тура отображаются корректно
- [ ] Форма бронирования и оплаты кликабельна, поля доступны
- [ ] Модальные окна/тосты не перекрывают контент
- [ ] Шрифты и иконки грузятся (нет «квадратиков»)
- [ ] DevTools → Console без критичных ошибок
- [ ] DevTools → Network: нет 4xx/5xx по ассетам
