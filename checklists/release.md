# Релизный чек-лист

## Перед релизом (staging)
- [ ] Все P1/P2 тест-кейсы выполнены
- [ ] Регрессионный чек-лист пройден по затронутым областям
- [ ] Нет открытых дефектов Blocker/Critical
- [ ] API-набор (Newman) зелёный
- [ ] Data-quality проверки без нарушений
- [ ] Release notes и затронутые области согласованы

## Во время релиза
- [ ] Деплой прошёл без ошибок (CI/CD)
- [ ] Миграции БД применены (если есть)

## После релиза (production)
- [ ] Smoke-чек-лист на production пройден
- [ ] Платёжный сценарий проверен на боевом окружении (контрольная оплата/возврат)
- [ ] Мониторинг/уведомления (Telegram-бот, n8n) активны
- [ ] Нет всплеска ошибок 5xx в логах
- [ ] План отката подтверждён на случай инцидента
